The javax.swing Library

Summary

Java's Swing library is used to create an interactible interface for the user, rather than a basic text-based one, or none at all. From windows and text to buttons and sliders, the Swing library is a good pick for programmers' needs when it comes to creating a visual interface for their programs.

Here is an example of a Swing window. There are several aspects to it; the text, the buttons, the scroll bar, and the window itself. These are all elements that can be found within the Swing library, under JFrame, JLabel, JButton, JScrollBar, and JToolBar. ImageIcon and BorderFactory can also be found here.

The AWT Class

The majority (if not all) of Swing's classes and methods come the AWT (Abstract Window Toolkit) class. The AWT class contains many visual interface related content, not unlike Swing. However, AWT's classes are abstract, and are meant to be modified in one way or another. Frame, Window, Label, and Image are all abstract classes within the AWT library. java.applet is another library that takes use of AWT's contents. However, unlike Swing, Applet uses AWT alongside other functions instead of completely basing itself off of it or recreating it.

JComponents & JContainers

Swing consists of JComponent objects and JContainer objects. JComponent objects have their own designated purposes, while JContainer objects, while they also have their own respective uses, are meant to contain JComponent objects or other JContainer objects.

Here is an example of how JContainers work. They can contain JComponent objects and JContainer objects alike. They can be added with the add() methods and revoked with the remove() commands.

JFrames, JLabels, and JPanels

JFrame, JLabel, and JPanel objects are extensions of either the JComponent class or the JContainer class. Since JFrame and JPanel objects can contain other components, they are subclasses of the JContainer class. Since JLabel cannot, it is simply a subclass of the JComponent class. These three Swing components are the most commonly used of the three. JFrame is used to create windows/frames, JLabel is used to create elements of text or images, and JPanel is used to catagorize and separate components from each other, not unlike the <div> HTML element.

More Swing Elements

The JButton, JScrollPane, JLayeredPane, and JToolBar classes are more Swing components that are less commonly used, but still used regardless. In visual interfaces, these are used very often. Swing interfaces can easily be compared to HTML. For example, a JButton can be compared to the <button> tag in HTML. JLabel with <label>. JTextField with <textfield>, and many more. In fact, Swing components, specifically JLabel, can support HTML text. If you were to type something like new JLabel("<html><b><u>Important text</u></b></html>");, you would get Important text as a result. Regardless, Swing and HTML have some similarities, so you could find some Swing components similar to HTML tags.
